# workspace setup

## 3 useful shortcuts

- Ctrl + P: This shortcut lets me quickly find and open any file in my project. Just type part of the file name, and it pops up! No need to scroll through folders.

- Ctrl + Shift + F: When I want to search for something like a function or variable across all my files, this is what i use. It saves me from having to open each file one by one.

- Ctrl + Shift + P: This opens the command palette. Where you can search for and run any command in VSCode.


## 3 impactful settings

- Format on Save: It automatically formats my code every time I hit Save (Ctrl + S). No more worrying about indentation or spacing issues.

- Tab Size = 2: This makes each tab only 2 spaces wide, which is standard in most JavaScript/TypeScript projects. It keeps the code neat and compact, especially when there are a lot of nested structures.

- Auto Save on Focus Change: I’ve got this setting so my files save automatically when I switch to another file or window. It prevents me from forgetting to save and losing any work.


## 2 extensions you rely on

- Prettier: This is the best code formatter ever. It automatically fixes all the small formatting issues in my code like spaces, line breaks, etc so everything looks consistent and professional.

- ESLint: This one helps me catch coding mistakes in JavaScript/TypeScript. It checks my code for errors and suggests fixes, which is awesome because it helps me write cleaner, error-free code.
